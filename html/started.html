Getting Started

Installing Libra

There are several ways to begin working with the API. Below, a step-by-step guide to different options are highlighted. If you're already familiar with python and PyPi, feel free to skip ahead and just do 

pip install libra

Setting up your local enviroment

  • Install Python 3.6+ directly onto your local machine. Anything lower then 3.6 WILL not work.
  • For Libra, and other libraries, pip is the best option for installing libraries. Install the most recent version.
  • If you're looking for an IDE we reccomend installing PyCharm
  • Now you can do pip install libra in your terminal. If you want to setup a local enviroment, follow this guide and then run the command.

    • Now you're ready to code! Open up a notebook/terminal put from libra import client at the top of your file and start working!

Libra is the nexus of modern machine learning

We've combined technologies from the most popular platforms to create a complete experience

  • Keras's model building techniques for improved modularity
  • TensorFlow's core computational power
  • PyTorch's Acceleration for optimization and efficiency
  • Sklearn's one-line approach to building and training models
  • Keras Tuner's intelligent neural network tuning strategies

Libra has a large community and many resources available

With a strong core team and mentorship opportunities, Libra offers many sockets to find success.

Join our slack group, where the founding developers are constantly asking questions about the library.

If you're an organization, fill out our form today and we'll hold either live or online instruction on how to use our tools.

Libra makes collaboration between experience levels possible

  • The API can be used by a complete novice, without modifying any parameters.
  • It can also be used by a moderately experienced developer, who understands the basics of the process.
  • It can also be used by a machine learning engineer, who understands the details of all the parameters offered by the library
  • This allows for users of all sorts of experiences collaborate on the same task.
