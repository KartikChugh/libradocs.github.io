Installation Directions

Step 0 of working with Libra is to install

Install Latest Release Version:

pip install -U libra

Install Directory from Github:

git clone https://github.com/Palashio/libra.git
cd libra
pip install .

We also add notes on the Github such as expected future release dates and version control. You can also manually install by visiting our Github and downloading the .zip file.

Install from Conda:

conda install libra -c conda-forge
